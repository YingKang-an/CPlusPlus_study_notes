# CPlusPlus-Learning-Notes  
> 个人 C++ 学习笔记与实战代码仓库，完全对标学习课程模块，涵盖 C++ 初识、面向过程到面向对象的核心知识点，同步配套实操代码与笔记解析。

---

## 🎯 项目简介
欢迎来到我的 **C++ 学习仓库**！  
本仓库严格按照学习课程的模块划分，记录了从 C++ 基础语法到面向对象核心特性的学习笔记、知识点梳理及对应实战代码，适合跟随课程节奏逐步掌握 C++ 编程能力。

---

## 🗂️ 项目结构
仓库依据课程目录分为 **C++ 初识（面向过程编程）** 和 **面向对象编程** 两大核心模块，每个模块下按课程知识点拆分子目录，代码与课程内容一一对应，便于查阅和实操。

---

### 1️⃣ C++ 初识（面向过程编程）
> 涵盖 C++ 基础语法、数据类型、程序流程控制等核心内容，是入门的基础环节。

| 目录 | 内容要点 |
|---|---|
| `01_运算符表/` | C++ 运算符分类、优先级与结合性笔记及示例 |
| `02_ASCII表/` | ASCII 码对照表及字符与 ASCII 码转换代码 |
| `03_C++初识/` | C++ 程序基本结构、编译运行流程、注释规范 |
| `04_基本数据类型-整型/` | 短整型、整型、长整型等整型类型的定义与使用 |
| `05_基本数据类型-bool/` | 布尔类型的取值、运算及应用场景 |
| `06_基本数据类型-浮点型/` | 单精度、双精度浮点型的精度与使用区别 |
| `07_基本数据类型-字符型/` | 字符型变量定义、转义字符的使用 |
| `08_运算符/` | 算术、赋值、比较、逻辑、位运算符的示例代码 |
| `09_分支跳转-if_switch/` | `if` 语句、`if-else`、`switch` 分支结构的实战 |
| `10_循环-while_for-do-while/` | `while`、`for`、`do-while` 循环的使用与嵌套 |
| `11_数组/` | 一维/二维数组的定义、遍历与数组越界规避 |
| `12_指针基础/` | 指针定义、内存地址操作、指针与数组的关联 |
| `13_字符串/` | C 风格字符串、字符数组的处理与常用操作 |
| `14_引用变量/` | 引用的定义、特性及与指针的区别示例 |
| `15_函数/` | 函数定义、参数传递、函数重载、默认参数与递归 |
| `16_结构体/` | 结构体定义、结构体数组、结构体指针的使用 |
| `17_内存存储/` | 栈内存、堆内存、静态存储区的特性与 `new/delete` 使用 |
| `18_其他杂项/` | 课程中零散知识点的整理与补充示例 |

---

### 2️⃣ 面向对象编程
> 聚焦 C++ 类与对象核心特性，涵盖类的定义、构造析构、继承多态等关键内容。

| 目录 | 内容要点 |
|---|---|
| `01_课程介绍/` | 面向对象编程思想及课程学习规划笔记 |
| `02_初识类class/` | 类的定义、对象实例化、成员访问权限 |
| `03_构造与析构/` | 构造函数、析构函数的定义与调用时机 |
| `04_运算符重载/` | 常见运算符的重载规则与实战代码 |
| `05_继承/` | 单继承、多继承的实现，基类与派生类的关系 |
| `06_多态/` | 虚函数、纯虚函数、动态多态的实现原理与示例 |
| `07_string类/` | C++ 标准库 `string` 类的常用接口（拼接、查找、替换等） |
